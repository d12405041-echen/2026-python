## 🤖 AI 協作與「四項原則」執行

### 1. 我問 AI 的 3 個問題
*   如何優化 Python 大量小檔案的磁碟 I/O？
*   在 Python 中實作「質數生成器」的最快方法？
*   如何寫出符合 Google Python Style Guide 的繁體中文註解？

### 2. 我採用的建議
*   **Generator (生成器)**：AI 建議在處理大數據（如質數篩選）時使用 `yield`，這大幅節省了記憶體佔用。
*   **`sys.stdin.read().split()`**：這是我這學期學到最強的技巧，AI 教我這樣寫可以無視任何空白與換行，精準讀取數字。

### 3. 我拒絕的建議
*   AI 建議我用 `pandas` 來處理 W02 的 Task 3。我拒絕了，因為我覺得大一基礎課應該用 **`collections.Counter`** 練習原生資料結構，而不是依賴第三方庫。

### 4. AI 錯誤修正案例
*   **錯誤案例**：AI 在寫質數生成器時，漏掉了 $N=2$ 的特殊情況判定。
*   **手動修正**：我手動加入了 `if n < 2: return`，確保邊界測試不會噴 Error。
