## 🤖 AI 協作紀錄

### 1. 我問 AI 的 3 個問題
*   如何去除清單中的重複項但同時保持原始順序？（針對 Task 1）
*   在 Python 中如何根據多個條件（如分數降序、年齡升序、姓名升序）進行複雜排序？（針對 Task 2）
*   如何同時統計使用者的總事件數與全域最常見的動作？（針對 Task 3）

### 2. 我採用的建議
*   **`sorted(key=lambda x: ...)`**：AI 教我使用 lambda 並在分數前加負號實作降序，這讓 Task 2 的多重排序邏輯變得極簡，符合老師「一打多」的要求。
*   **`collections.Counter`**：針對 Task 3，採用 AI 建議的內建計數器來抓取 `most_common(1)`，效率遠高於自己寫迴圈找最大值。

### 3. 我拒絕的建議
*   **拒絕 `set()` 去重**：針對 Task 1，AI 一開始建議用 `list(set(data))`。但我讀過 `HOMEWORK.md` 聖經，發現這會打亂題目要求的「原始順序」。我拒絕了這個建議，改用 `dict.fromkeys()` 來兼顧去重與順序。
*   **拒絕 `zip_longest` 補位**：在考慮 Task 3 的空輸入時，AI 建議了一些複雜的預填充邏輯。我覺得太噁心（過度工程），堅持用 `defaultdict(int)` 的自然屬性來處理。

### 4. AI 錯誤修正案例
*   **排序優先級錯誤**：AI 在 Task 2 最初給的建議是分兩次 `sort()`，但我識破這在 Python 的穩定排序中雖然可行，但代碼不夠精煉。
*   **手動修正**：我將其整合為單一 `sorted()` 呼叫，並正確處理了 `age` 升序與 `score` 降序的衝突。
